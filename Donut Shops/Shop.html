How the Program Works:

  1. Using the data that we alredy know for each location, we create methods to plan donut production for any given day.
  2. Before we start inputting the parameters, I created a couple global functions that we could call upon later: myRandomNum, getCustPerHour, getDonpHr, getTotal.
  3. Once we have or global functions, it's time to create a Shop object tthat provides us a place to pass each location's data.
  4. Within the Shop object, more functions were added with parameters specific to the shop's location.
  5. The Menu bar at the top will eventually allow you to click through each location for their production numbers; while the "All Locations" button will provide you with the default summary page for all locations.
